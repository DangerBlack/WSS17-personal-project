Complete the Function

Complete the Function is a game designed by DangerBlack during the Wolfram Summer School 2017, the aim of the game is: given an expression in Wolfram Mathematica Language finds the missing sport and complete with the better function.
The game is enterely developed trought Mathematica 11.1 during a 3 weeks stage in Boston, Ma.

Each expression contain one or more missing spot must be completed with the appropriate function.
In easy mode the game give you a Tip about the size of the missin function.
In hard mode the game give you more than one mising spot to replace.
